如何消除块之间的神秘空白

inline-block块之间有空隙的原因是：html源码中块之间存在空白符(换行符、空格、缩进)

html中多个空白符在浏览器中只占一个空白符的大小
空白符：| |(←这里有很多个空格)
要想使多个空格生效需要用&nbsp;和空格分隔：|     |(←这里有很多个空格)

inline-block + font-size: 0;

233

优点：兼容性好，破坏性小

缺点：子元素需要将fontSize重置

.parent2 {
  font-size: 0;
}
.parent2 .child {
  display: inline-block;
  font-size: 14px;
  overflow: hidden;
}
